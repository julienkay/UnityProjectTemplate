# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/en/1.0.0/)
and this project adheres to [Semantic Versioning](http://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-14

### Added

- Package and companion project creation wizards for scaffolding UPM-ready repository layouts.
- Template generation for package manifests, asmdefs, assembly info, readmes, changelogs, sample scripts, and docfx documentation files.
- Dependency editing with manifest-style list controls and package autocomplete sourced from Unity and scoped registries.
- Supporting editor utilities for repository initialization and asset reserialization.
